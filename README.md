# 【Excel VBA】ファイルリスト作成ツール
指定したフォルダ内の全ファイルを、サブフォルダまで含めて
一括リストアップするツールです。

## 必要環境
- Microsoft Excel 2016以降
- Windows 10 / 11

## 使い方
1. `FileList_Tool.xlsm` を開く
2. 「設定」シートでスキャン対象のフォルダを選択
3. 「リスト作成」ボタンをクリック
4. 結果シートにファイル一覧が自動出力される

## 主な機能
- 再帰処理によるサブフォルダの全階層スキャン
- FileSystemObjectによる存在チェック・エラー処理
- 1行おきの着色（ゼブラ設定）
- オートフィルターの自動付与
- フォルダ参照ダイアログによるGUI操作

## 導入効果
手作業では1,000件あたり約3時間を要するファイル棚卸し業務を
ボタン一つで数秒に短縮します。
